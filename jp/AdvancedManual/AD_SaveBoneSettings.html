ボーンの設定をエクスポート/インポート


1.0.0

骨の設定のエクスポート




(1) メッシュグループの 「Bone」タブで、(2) 「Export / Import Bones」ボタンをクリックします。




(3) 「Export」ボタンをクリックすると設定がファイルとして保存されます。




骨の設定のインポート




骨の設定をインポートする新しいメッシュグループを準備します。
もう一つのapPortraitが可能です。




(1) 「Bone」タブを選択します。
(2) 「Export / Import Bones」ボタンを押します
(3) 「Load File」ボタンを押すと、
(4) 保存したファイルを開きます。




1. Export : ファイルとして保存
2. Load File : ファイルを開いて、インポートを準備し
3. Bone Name : 骨の名前
4. Import : インポートするかどうか
5. IK : IK設定のインポートするかどうか
6. Shape : Shapeの設定をインポートするかどうか
7. Enable/Disable : すべての骨を選択または選択解除
8. Import Scale : 保存されたデータのサイズを変更する割合
9. Import Button : 骨のデータをメッシュグループにインポート




上の画面のように、ファイルの骨のデータをインポートすることができます。






上記のように一部のボーンを除外したり、シェイプをインポートから除外することもできます。






< 意図したとおり、IKが動作するように誘導 ボーンをミラーコピー >
