인스펙터 UI


1.3.4



유니티 에디터에서 AnyPortrait로 만든 apPortrait 오브젝트를 선택하면, 인스펙터(Inspector)에 그 정보가 나타납니다.
이 페이지에서는 인스펙터의 화면 구성에 대해서 설명합니다.


1. Open Editor and Select : 에디터를 열고 바로 현재 오브젝트를 편집할 수 있도록 선택합니다. (Optimized Bake된 경우 에디터를 열 수는 있지만 편집은 불가능합니다.)
2. Quick Bake : 바로 Bake를 합니다.
3. Refresh Meshes : Bake된 캐릭터의 메시들을 갱신합니다.
3. Is Important : 이 설정이 켜지면 게임의 모든 프레임에서 업데이트 되고, 설정이 꺼진 경우 적은 FPS에서만 업데이트 됩니다. 구동하고자 하는 FPS는 설정 다이얼로그나 Bake 다이얼로그에서 변경할 수 있습니다.
5. Sorting Layer/Order : 출력 순서를 변경합니다.
6. Billboard Type : 빌보드 설정을 변경합니다.
7. Root Portraits : 루트 유닛들이 나타납니다.
8. Sockets : 본 또는 메시의 소켓들이 나타납니다. (v1.3.4)
9. Animation Clips : 애니메이션들이 나타나며, 메카님을 사용할 경우 이에 대응하는 실제 Animation Clip 에셋이 같이 보여집니다.
10. Empty Animation Clip : 메카님을 사용하는 경우 "Empty State"에 해당하는 Animation Clip 에셋입니다.
11. Mecanim Settings : 메카님 사용 여부와 Animator 및 Animator의 레이어들이 나타납니다. Refresh Layers 버튼을 눌러야 갱신됩니다.
12. Timeline Settings : 유니티의 타임라인과 연동할 때의 설정들입니다.
13. Animation Events : 애니메이션 이벤트입니다. 이벤트 타입에 따라 리스너 및 콜백 함수를 지정할 수 있습니다. (v1.3.4)
14. Control Parameters : 컨트롤 파라미터들이며, 바로 슬라이더를 움직여서 확인해볼 수 있습니다. 게임이 실행 중이 아닐 때에도 테스트해볼 수 있습니다.
15. Show All Properties : apPortrait의 모든 프로퍼티를 열어서 확인할 수 있습니다. (데이터가 많으므로 에디터의 성능이 느려질 수 있습니다.)
