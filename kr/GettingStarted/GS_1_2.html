1.2. 이미지 로드하기


1.0.0



Hierarchy UI에서 (1) "Add Image" 버튼을 누르고 생성된 이미지를 선택합니다.
이미지를 선택하면 현재는 아무것도 나오지 않습니다. 에셋들 중에서 이미지 리소스(Texture)를 열어서 지정해야합니다.
(2) Select Image 버튼을 눌러서 이미지 선택 다이얼로그를 호출합니다.
AnyPortrait 패키지를 설치하면 같이 포함되어 제공되는 (3) 슬라임 이미지를 선택하고, (4) Select 버튼을 누릅니다.




이미지 에셋을 로드하면 우측의 상세 정보 UI에 자동으로 정보가 채워집니다.
