Animation Curve Presets


1.0.0



There are 4 curve preset buttons at the bottom of the UI for setting animation curves.
Presets that can be applied to frequently used shapes for smooth type curves.
By using these buttons, you can shorten the time to edit the curve every time.




The 4 presets are as follows.
1. Default : When you create a keyframe, the first is the shape of the curve.
2. Acceleration and Deceleration : It is a curve that accelerates at first and decelerates at the end. Accelerating and decelerating more than the default value.
3. Acceleration : An acceleration curve that gradually changes to the shape of the next keyframe.
4. Deceleration : Deceleration curve that changes rapidly at first, then slows down slowly.
