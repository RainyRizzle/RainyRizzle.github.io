Optimized Bake


1.0.0

Moving a character in the AnyPortrait editor to the scene is called Bake.
"Data used in the editor" and "Data executed in the scene" are different. Therefore, a bake process to move data is essential.
If all the edits are finished, the "Data used in the editor" is no longer needed.
So if you only have Bake that only "Data executed in the scene" exists, you can save memory and prefab capacity.
This page covers Optimized Bake.




(1) Open the Bake dialog box.
(2) Click the Optimized Bake button and execute optimized Bake.
There is a Target field on the button.
You can bake it with the specified target.
If the Target is empty, create a new object.
The object you are working on in the editor can not be specified as Target.




You can see a new character in the scene with the name "(Optimized)" in the name.
This object has only "Data executed in the scene".




After optimized bake, it is automatically connected to Target.


Note: The AnyPortrait editor distinguishes between "optimized apPortrait" that can not be edited.
Therefore, these objects are not exposed to the editor.
